# Teste Econverse - Vaga Desenvolvedor Front-End

Este repositório contém a solução para o desafio técnico da Econverse.

## 🚀 Tecnologias Utilizadas

- React (com Hooks como useState e useEffect)
- TypeScript (para tipagem estática e segurança do código)
- SASS (Pré-processador para estilização avançada)

🛠️ Funcionalidades Implementadas

- Vitrina Dinâmica: Consumo de dados via `fetch` a partir de um ficheiro JSON.
- Modal de Detalhes: Interação que abre um modal com informações específicas do produto ao clicar.
- Newsletter: Formulário de inscrição com validação básica.
- Design Responsivo: Adaptado para diferentes resoluções conforme o layout do Figma.

## 📦 Como rodar o projeto localmente

Siga os passos abaixo para executar o projeto na sua máquina:

1. Clonar o repositório:
   ```bash
   git clone https://github.com/RaulSilvaB/econverse-frontend-test.git
   ```
